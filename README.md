### Installation

1. Clone the repository

```bash

cd personal-website
```

2. Install dependencies

```bash
bun install --frozen-lockfile
```

### Development

Start the development server:

```bash
bun run dev
```

The application will be available at `http://localhost:3000`

### Production

Build for production:

```bash
bun run build
```
